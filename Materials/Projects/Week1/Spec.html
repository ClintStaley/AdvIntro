Week 1

Assignments

This assignment uses PyGame to draw different pieces of abstract art. The point is to exercise basic coding concepts such as lists, loops, if-statements, and functions, but it's a more entertaining way to do so than, say, checkbook-balancing code.

Basic

AbstractGridBa.py

Write code to generate the AbstractGridBa.png image. Start with the supplied AbstractGridBa.py file, which offers initial data structres, e.g. giving the colors and the rectangular areas in the image, to save you the tedium of figuring those out.

Importantly, you're allowed only one window.set_at call to set one pixel to a given color -- no other graphical calls. So, you'll need to do a nested loop across the range of X/Y coordinates in the 800x800 image, and determine what color each should be.

Note that AbstractGridBa.png is gray unless your point is within one of four rectangular regions. Within those, it is black if it's in a 40x40 "checkerboard" square that's black. (Modulo (%) may be of use here.) And within the rectangles, and within white squares, pixels are colored in 100-pixel rings radiating out from the image center.

You may find it easiest to this incrementally. Start, for instance, by getting the background gray, and each rectangle white. Then add the black squares, and finally the circular rings

You must write this reasonably elegantly, and with correct style. My own implementation has a draw_pattern method of under 20 lines not counting blank lines and comments.

Bronze

AbstractPatternBr.py

Write code to generate AbstractPatternBr.png. Do this by making functions for the following:

1. Drawing a shaded "bullseye" circle on white background within a rectangular area

2. Drawing a grid of circles within a rectangular area, allowing a choice of solid color or bullseye (the latter by calling function 1)

Silver (no requirement)

Gold

AbstractPatternGo.py

Add one more method to draw a spiral in a rectangular area, and use this to create AbstractPatternGo.png. The spiral bands are 4 pixels wide, with 4 pixel gaps between. Yours may be slightly different from mine, but must be a true continous spiral filling the region, starting upper left and turning clockwise.

Platinum

AbstractPatternPl.py

If there is enough interest, say at least 3 students, I'll run a Code Art competition for extra credit. This competition involves writing a short, elegant bit of code, at most a page or two, that produces an aesthetically interesting abstract artistic pattern. Entries are judged on the combination of code elegance and cleverness, and on the beauty of the output. If we run this, I'll volunteer to add an entry of my own as well.
