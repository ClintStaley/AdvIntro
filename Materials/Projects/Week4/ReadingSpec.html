Questions to work with.

1. Consider that package statement on line 1. What *is* a Java package, and why are they useful? Why did I preface the package name with com.cstaley, instead of just, say using "package MMind"? if you were setting up your own package, what package names could you use, by convention?

6. What is "Character"? An object? a package? a class? What methods does it supply?

7. What is the difference between a while and a do-while loop?

8. What's up with line 64? If I write bad = bad || test, what is the effect? Why do we preface the right side with "bad ||"?

9. What happens if I omit the !bad on line 59?

10. What code should be filled in on line 68? (Do this w/o an LLM's help)

11. On line 60, what does the nested lineNdx++ do? if lineNdx == 2, for instance, will charAt receive argument 2 or 3?
