Java LLM Study 1



Review the LLMStudy file provided at the top level of the course. Following its guidelines, and using Claude, build an LLM study for these Java Topics, covered in Guess.java and Model.java.

List of topics:

1. Packages and package naming

2. Import statements

3. Record classes

4. Pseudorandom number generation and the Random class

5. Elemental types -- all of them including the varieties of int and float.

6. ++ and -- operators, in both pre- and post- forms



And a reminder for this and all future LLM Studies: you need deep and interactive exploration of these topics, per the LLM Study Guide. Quizzes based on the study will anticipate this. Here's a brief couple of examples:



Not a good quiz question: "What's a record class?"

Good quiz question and good LLM question: "Rewrite the following to produce an equivalent class without using record: record Sample(int x, float y)"



Not a good quiz question: "What does this assignment do, assuming x and y are 1? z = x++ + y++"

Good quiz question and good LLM question: "What happens if I use an autoincrement on the same variable twice? z = a[x++] + b[x++]"
