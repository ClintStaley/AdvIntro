LLM Study: Trees. Sets and Maps

Queues

For this final LLM-reading assignment, cover these items our schedule didn't allow in class.

1. Impementation of tree operations, especially node deletion, which takes a bit of cleverness.

2. The Set and Map interfaces in java.util, and how trees can implement each of them.

3. The idea that Map values(), entrySet(), and keySet() values return an object that does *not* copy, and the notion of a "view" interface.

4. The Comparator and Comparable interfaces, and how they relate.

5. How one might use a Stack to implement TSItr, the iterator for a TreeSet.

6. The code I supplied in TreeSet.java and TreeMap.java, including preparation to implement some of the missing methods in those classes.



For each of these six conversations, the point is to use the LLM as an interactive textbook or tutor, not just an answer-machine. In particular, your conversation must include:

1. Deeper discussions about each concept, not just a one shot question/answer.

2. Extending on and/or improving the code that was supplied, where relevant.

3. Getting and answering a quiz from the LLM on each of these.

4. Catching the LLM in a significant error, not just some minor syntax or terminology quibble. (I'll settle for doing so in 2 of the 6 topics)

Turnin

Submit your whole chat as Chat.pdf, along with the LLMEval.pdf file.
